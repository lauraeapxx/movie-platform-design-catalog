# movie-platform-design-catalog
该项目围绕在线电影与电视剧平台的页面设计结构展开整理，对导航逻辑层级、模块排列顺序以及播放页信息区块构成进行系统记录。通过结构化文字归档建立资料库，用于研究不同影视网站在页面组织与内容展示方面的演变趋势。仓库仅保存公开页面资料，不包含影视资源内容或在线播放功能。
